# **App Name**: Thru

## Core Features:

- Consumer Order Management: Consumer dashboard for placing and tracking orders.
- Vendor Order Management: Vendor backend to accept/reject orders and track customer arrivals.
- Real-time Availability Display: Display product listings with real-time availability from local vendors.
- Arrival Time Prediction: AI-powered ETA prediction tool that estimates customer arrival time based on current traffic and location, communicated to vendor backend.
- Push Notifications: Real-time notifications for order updates and arrival alerts.

## Style Guidelines:

- Primary color: #F6675E.
- Background color: Light gray (#F0F0F5), very lightly tinted with #F6675E, for a clean backdrop.
- Accent color: Muted teal (#70A1AF) to highlight calls to action.
- Body and headline font: 'Inter', sans-serif, for clear readability and a modern aesthetic.
- Use minimalist, line-based icons to represent different product categories and actions.
- Clean, card-based layout for product listings and order summaries.
- Subtle transition animations and loading spinners to enhance user experience.
